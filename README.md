# Md Kashif Portfolio Website

A modern, fully animated personal portfolio website built with React, TypeScript, and Tailwind CSS.

## Requirements

- Node.js (version 16 or higher)
- npm (comes with Node.js)

## How to Run on Your PC

### Step 1: Download and Extract
1. Download the `mds-portfolio.zip` file from Replit
2. Extract the zip file to a folder on your computer
3. Open terminal/command prompt and navigate to the extracted folder

### Step 2: Install Dependencies
```bash
npm install
```

### Step 3: Start the Development Server
```bash
npm run dev
```

The website will open automatically in your browser at `http://localhost:5000`

## Project Structure

- `client/` - React frontend application
- `server/` - Express.js backend server
- `shared/` - Shared TypeScript types and schemas
- `attached_assets/` - Images and other assets
